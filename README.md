# ⚡ Smart Grid Monitoring System

## 📌 Overview
A Python-based IoT system designed to monitor and detect critical power grid issues in real-time.

## 🚨 Problem Addressed
Frequent rural grid failures due to:
- Overheating
- Power theft
- Pole tilting
- Wire cuts

## 💡 Solution
This project detects faults using sensor-based logic and sends real-time alerts for preventive action.

## 🛠 Technologies Used
- Python
- ESP32
- IoT Sensors
- ThingSpeak Cloud
- Thonny IDE

## 📂 Project Modules
- OverHeatingFin.py – Overheat detection
- ThefFin.py – Power theft logic
- Tilt.py – Tilt monitoring
- wirecut20.py – Wire cut detection

## 🔮 Future Improvements
- Relay-based automatic power cut-off system
- AI-based fault prediction and anomaly detection
